# Reproduction bundle — ICML 2026 paper 32495

This bundle contains the exact scaled-proxy implementation, the timestamped K-LIVE audit, raw trial outputs, summaries, generated figures, poster, and source-paper snapshot used in the Trackio logbook.

Scientific scope: the released paper code and K-LIVE repository were unavailable during the audit, so Claims 1–2 are tested with Qwen2.5-Coder-1.5B on three sklearn classification tasks and Claim 3 is a consistency/URL/cutoff audit. These are labelled proxy or audit evidence, not a full 4,000-run Kaggle replication.

After publication, download the complete bundle from the Bucket link in the logbook’s Conclusion page. The Hub dataset mirror at https://huggingface.co/datasets/binzhango/icml-32495-reproduction-artifacts contains the GPU Job script and raw GPU outputs.
