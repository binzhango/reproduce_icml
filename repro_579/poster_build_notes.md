# Poster build notes

- Content inventory: benchmark-scale table, construction flow, Table 2 score table, threshold audit, deterministic proxy statistics, GPU toy results, and scope/conclusion. No single dominant paper figure was available, so the compact two-column portrait scaffold was selected and widened to a 30 × 36 inch custom canvas to prevent crowding.
- Palette: cool blue derived from the blue/orange visual language used throughout the paper figures; Posterly's neutral blue token set already matched, so the cool blue accent and warm gold evidence highlight were retained.
- Framing: faithful verification. Paper-reported values are distinguished from independently computed Table 2 counts and from the toy proxy. No toy score is presented as a Squirrel score.
- Figures: the poster is data/table-driven; the optional real-paper-figure asset gate is not used because the requested reproduction centers on numeric claims and the public benchmark data are unavailable.
- Header: no fabricated affiliation logo or QR code. Authors and affiliation/reproducer context remain on one line beneath the title.
